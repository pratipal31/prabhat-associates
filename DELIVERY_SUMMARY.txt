🚀 QUICK START (5 MINUTES):
================================================================================

1. Install Dependencies:
   $ cd /vercel/share/v0-project
   $ pnpm install

2. Run Locally:
   $ pnpm dev
   Open: http://localhost:3000

3. Make Changes:
   Edit any file in /components or /app
   Auto-reload works!

4. Deploy:
   Option A: vercel --prod
   Option B: Push to GitHub → Connect to Vercel
   Option C: Docker/traditional server

5. Customize:
   Update company info, change colors, replace logo
   (See GETTING_STARTED.md for detailed steps)
